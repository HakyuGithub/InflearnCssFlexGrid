-------------------------------------------------

flex-basis
=> 플렉스 아이템의 메인축에서의 기본크기를 설정한다.
=> width, height에 넣는 수치단위를 넣을 수 있다.
=> auto가 기본값 (해당 요소의 width나 height를 채택)

-------------------------------------------------

flex-grow
=> 플렉스 아이템이 flex-basis보다 커질수 있는지를 결정
=> 숫자값이 들어가서 flex-basis를 제외한 여백 부분을 
   해당 값만큼 나눠가진다.
=> 0이 기본값

-------------------------------------------------

flex-shrink
=> 플렉스 아이템이 flex-basis값보다 작아질 수 있는지를 결정
=> 숫자값이 들어가서 flex-basis를 제외한 여백 부분이
   줄어드는 비율을 명시한다.
=> 1이 기본값

------------------------------------------------- 

단축형
  flex: flex-grow flex-shrink flex-basis;

flex: 1;
  => 1 1 0%;

flex: 1 500px;
  => 1 1 500px;

-------------------------------------------------

flex-grow, flex-shrink를 적용해도 정확한 비율의 
다단 컬럼 레이아웃을 구성하기 어렵다.

item에 flex를 대략적으로 설정하고
각 아이템에 직접 width를 주어 설정하면 명확하게 다단컬럼
작성가능하다.

-------------------------------------------------

align-self
=> align-items는 컨테이너에 설정되어 전체 아이템에 적용된다.
=> 개별 아이템에 적용될 수직축 정렬을 설정

-------------------------------------------------

order
=> 아이템의 시각적 나열 순서를 의미
=> 작은 숫자가 할당될 수록 먼저 배치

-------------------------------------------------
